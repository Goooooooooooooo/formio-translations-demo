Platform Resource

Formio.js uses i18n, a lightweight translation module, to allow users to pass in a translation option during form construction. During this process, users can hook into a form.io resource that contains a custom translation object. The result, an in-portal interface that can control form translations.

PhraseApp Integration

PhraseApp is an automated localization processes that empowers users to edit language files via an in-context editor. This demo showcases how users can overlay the aforementioned PhraseApp service on top of an Angular 2+ application.
