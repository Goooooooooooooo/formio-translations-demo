Portal - A Translation Resources

This example showcases Form.io's ability to store translation definitions within a remote resource on portal.form.io. Users can leverage this functionality to transform a resource within their remote project to serve as an external or embedded interface to manage and create translations. The image shows the resource state which is responsible for both managing the which language in used in addition to which phrases are translated within the form.

As a reminder, you can download a copy of this language resource or you can locate a copy in the assets folder of this repository. If you are interested in replicating this environment, learn more by visiting our help docs.
